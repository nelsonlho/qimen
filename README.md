# 奇門遁甲排盤(陽盤・置閏法)

時家轉盤(陽盤)奇門遁甲排盤程式。網頁版,核心演算法與界面分離,便於日後行動裝置(React Native / Capacitor)復用。

## 起局法度

- **定局:置閏法(超神接氣)**
  - 符頭:甲、己之日。子午卯酉日為上元,寅申巳亥為中元,辰戌丑未為下元。
  - 符頭先節氣而至為「超神」,後至為「接氣」,同日為「正授」。
  - 符頭超節氣 **九日(實差,即節氣日 − 符頭日 ≥ 9)** 以上,逢**芒種、大雪**置閏——重複其上中下三元一次。門檻可經 `QimenOptions.leapThreshold` 調整。
  - 實作:自 1860 年起逐三元段推演至 2105 年。初始指派縱有一節之差,首次置閏後必自行收斂(相差十五日之兩態,超者先閏,閏即合流),故 **支援範圍約 1900–2100 年**。
- **節氣**:採 `lunar-typescript` 天文法真節氣(中國標準時間 UTC+8)。輸入時間一律視為東八區。
- **盤式**:轉盤。天禽寄坤二隨天芮;中五宮干寄坤二。八神:值符騰蛇太陰六合白虎玄武九地九天,陽順陰逆。值使隨時辰行九宮(含中五),落中五寄坤二。
- **子時換日**:預設夜子時(23 時)起新日,可切換零時換日。
- **九星旺衰**:依《煙波釣叟歌》「與我同行即為相,我生之月誠為旺,廢於父母休於財,囚於鬼兮真不妄」——同行相、我生旺、父母廢、財休、鬼囚(按月令五行)。
- **斷析**:門宮生剋(迫、制、比和、宮生門、門生宮)、天地盤干十二長生(四維宮二支取旺為主)、十干剋應八十一格、六儀擊刑(以本宮天盤儀論,中五寄干隨禽而行者不論刑)、奇儀入墓、玉女守門、伏吟反吟。
- **可選盤層**:
  - **暗干**:「地甲為直符」——甲起旬首儀地盤宮,十干依序陽順陰逆飛佈,逢癸滿局復歸本宮(與甲同宮)。
  - **隱干**:時干(甲用旬首儀)加值使門落宮,循三奇六儀次序陽順陰逆飛佈(含中五)。
  - **帶干**:八門各攜當局地盤中本宮之干(如陰遁八局休門帶丙)。

## 校驗

- 刊行置閏年表:1975/78/81/84/87/90/98/2001/04/07 大雪閏、1993/96 芒種閏,模擬全數吻合。
- 刊例:2004-09-01 癸未日,置閏法白露上元**陰遁九局**(拆補法則處暑陰一局)——吻合。
- 全盤人工覆核一例(2026-07-05 巳時):地盤、天盤轉序、值符值使、八神、空亡、馬星皆合。
- `npm test` 含上述迴歸試。

已知流派歧異:置閏門檻有「實差九日」與「首尾兼計九日(實差八日)」兩說,個別年份(如 2010、2013)閏在芒種抑大雪因此而異。本程式從古法「超神不過九日」之實差九日說。

## 開發

```bash
npm install
npm run dev    # 開發伺服器
npm test       # 測試
npm run build  # 產出 dist/(含 PWA:manifest + service worker)
```

## 手機安裝(PWA)

`npm run build` 後將 `dist/` 部署至任何 HTTPS 靜態主機(或區網試用:`npm run dev -- --host`)。
手機瀏覽器開啟後:iOS Safari「分享 → 加入主畫面」;Android Chrome「安裝應用程式」。
即有朱印圖標、全屏無瀏覽器框、離線可用(思源宋體亦入緩存)。
日後欲上 App Store / Play Store,以 Capacitor 包殼即可,代碼不必改。

## 結構

- `src/core/` — 純 TypeScript,無 DOM 依賴(他日手機版直接復用)
  - `ganzhi.ts` 干支、五鼠遁、旬首、空亡、驛馬
  - `solarTerms.ts` 節氣(lunar-typescript 封裝,naive ms 表示)
  - `ju.ts` 置閏法定局模擬
  - `pan.ts` 排盤(地盤、天盤、八門、九星、八神)
- `src/App.tsx` — React 界面(響應式,兼手機瀏覽)
